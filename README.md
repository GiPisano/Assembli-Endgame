# 🎮 Assembly: Endgame (React)

A Hangman-style word guessing game built with React.
The goal is to guess the hidden programming-related word before running out of attempts.

## 🚀 Features

- Guess letters using an interactive keyboard
- Dynamic word display that reveals correct letters
- Incorrect guesses eliminate programming languages
- Win and lose game states
- Restart game functionality
- Clean and simple UI

## 🛠 Technologies Used

- React
- Vite
- JavaScript
- CSS

## 🎮 How to Play

- Guess the hidden word by clicking letters on the keyboard.

- If the letter is correct, it will appear in the word.

- If the letter is wrong, a programming language will be eliminated.

- Guess the entire word before running out of attempts to win the game.
